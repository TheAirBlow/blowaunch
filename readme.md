## 🎮 About Blowaunch
This is crossplatform open-source Minecraft launcher, written in C#.
Runs 1.7.2 and newer versions without any problems.
**Curretly in development!**

## 🚀 Getting Started
Download the latest release from [here](https://github.com/TheAirBlow/blowaunch/releases),

## 🎨 Features

#### Implemented features:

- Automatically downloads the required Java version, no need to download it for yourself.
- It's very easy to install Fabric or Forge - just select it in Modding dropdown!
- Easily manage multiple accounts and switch between them.

#### Some of the features we are still working on are:

- Built-in auto-updater. The launcher will always keep itself updated to the latest release.
- Manage your Minecraft profile directly from the launcher
- Install any mods from Modrinth and Curseforge with ease
- Install Curseforge/ATLauncher/etc. modpacks with ease
- Implement instances
- Liteloader support
- A lot more...
